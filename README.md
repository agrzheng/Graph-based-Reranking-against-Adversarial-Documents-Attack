# GRADA
This the repository of [GRADA: Graph-based Reranking against Adversarial Documents Attack](https://arxiv.org/abs/2505.07546) based on the previous paper: [PoisonedRAG: Knowledge Poisoning Attacks to Retrieval-Augmented Generation of Large Language Models](https://arxiv.org/abs/2402.07867) [Github Repo](https://github.com/sleeepeer/PoisonedRAG).

Check enviroment.yml for running enviroment.

The setup process will be similar to [Github Repo](https://github.com/sleeepeer/PoisonedRAG).
